# Griffin M1

## Overview
Medium-class lander delivering modular cargo. Reconfigured after VIPER cancellation to carry commercial and NASA payloads (Task Order 20A).

## Details
*   **Lander:** [Griffin](../landers/GRIFFIN.MD)
*   **Target Site:** Mons Mouton (South Pole region).
*   **Transfer Trajectory:** 3 - 33 Day Direct Cruise.
*   **Connectivity:**
    *   **Direct-to-Earth:** Actuated High-Gain X-Band antenna for high-rate surface telemetry.
    *   **Surface Relay:** 802.11n Wi-Fi for rover-to-lander communication.
*   **Precision:** Targeted accuracy of **100m x 100m** using the **OPAL** sensor.
*   **Payloads:**
    *   **NASA CLPS (Task Order 20A)**
    *   **[FLIP](../rovers/ASTROLAB_FLEX.MD)** (FLEX Lunar Innovation Platform) - Venturi Astrolab (Heavy Rover)
    *   **[CubeRover](../rovers/CUBEROVER.MD)** (2U Mobile Rover) - Astrobotic
    *   **BEACON** (Autonomous Navigation Demo) - Mission Control/Astrobotic
    *   **Nanofiche GLPH** (Galactic Library Preserve Humanity) - Arch Mission
    *   **MoonBox** (Keepsakes) - DHL/Astrobotic
    *   **Commemorative Plaques** - Nippon Travel Agency/Japanese Children
*   **Status:** Planned. Targeting the lunar South Pole. Mission profile optimized for heavy rover egress via dual ramps.
