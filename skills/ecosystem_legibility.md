# Ecosystem Legibility
## Skill Intent
Translate ongoing work and decision into legible signal that allow governance and contributors to understand progress without additional context. 
## Inputs
-  Reprository commits and documentation
- Project discussion and decisions
- MoonDAO governance milestones
## Outputs (Progress signals)
- Narrative surmarries aligns with recent commits.
- Clear  articulation of what changed, why it matters, and next steps.
- Reference to commits or PRs where applicable.
## Success criteria
- Governance can assess progress asynchronously.
- Narrative aligns with repository state.
- Reduced clarification request from reviewers.
## Review cadence
- Reviewed via pulled request comments.
- Iterated based on governance feedback.
