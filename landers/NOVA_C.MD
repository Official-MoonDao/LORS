# Nova-C

## Overview
Nova-C is a class of lunar lander designed by Intuitive Machines to deliver small commercial payloads to the lunar surface. It uses a single VR900 main engine running on Methalox (Liquid Methane and LOX).

## Key Features
*   **Propulsion:** Methalox engine, allowing for potential in-situ refueling in the future.
*   **Landing Tech:** Autonomous hazard detection and avoidance.
*   **24/7 Data:** Provides continuous data coverage for payloads.

## Missions
*   **IM-1 (Odysseus):** Successfully landed in Feb 2024 (Malapert A).
*   **IM-2 (Athena):** Targeted for South Pole.
*   **IM-3:** Targeted for Reiner Gamma.
