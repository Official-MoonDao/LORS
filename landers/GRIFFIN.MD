# Griffin

## Overview
Griffin is a medium-class lunar lander developed by Astrobotic. It is significantly larger than the Peregrine lander and was originally selected to deliver NASA's VIPER rover (mission canceled/replanned, but lander continues).

## Key Features
*   **Ramp:** Features an egress ramp designed for heavy rovers (up to 500-600kg).
*   **Precision:** Uses optical terrain relative navigation for safe landing (100m accuracy).
