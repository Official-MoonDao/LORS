# TENACIOUS

## Overview
TENACIOUS is a micro-rover developed by ispace's European subsidiary (Luxembourg). It is flying on the HAKUTO-R Mission 2.

## Key Features
*   **Structure:** Carbon Fiber Reinforced Polymer (CFRP) frame to minimize mass.
*   **Mission:** Demonstration of regolith collection (ownership transfer to NASA).
*   **Design:** Optimized for the HAKUTO-R lander's payload bay.
