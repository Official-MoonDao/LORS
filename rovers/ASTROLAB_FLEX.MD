# FLEX

## Overview
The Flexible Logistics and Exploration (FLEX) rover is a heavy-duty, multi-purpose vehicle designed by Venturi Astrolab. It is intended to function as a "pickup truck" for the Moon, capable of transporting diverse payloads using a modular interface.

## Key Features
*   **Modular Payload Interface:** Can pick up and drop off standardized cargo containers.
*   **Capacity:** designed to carry up to 1,500 kg of cargo.
*   **Versatility:** Can be operated remotely or by astronauts (LTV configuration).

## Missions
*   Planned for launch on SpaceX Starship (NET 2026).
