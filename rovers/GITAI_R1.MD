# GITAI R1

## Overview
GITAI R1 is a robotic rover designed for lunar infrastructure construction and maintenance tasks.

## Key Features
*   **Robotic Arms:** Equipped with "grapple" end-effectors for tool manipulation.
*   **Tasks:** Solar panel assembly, mining inspection, cable connection.
*   **Autonomy:** Designed for tele-operation and semi-autonomous tasks.
